Course outline:

  • Basic architecture of the ATMEL processors (1 week)
    • instruction set architecture
    • memory and I/O
    • general organization of on-chip I/O devices
  • Assembly language programming (1 week)
    • addressing, operations
    • timing considerations
    • the parallel port
  • Review of C (1 week)
    • basic control structures
    • simple code management (makefiles)
    • interfacing between C and assembler
  • On-chip components (1 week)
    • ports, timers
    • interrupts, interrupt programming, sleep modes
  • Serial peripherals (1 week)
    • serial communication --- SPI, IIC
    • bit-level protocols
  • Basic circuits (1 week)
    • Ohm's law, capacitance, RC circuits
    • Kirchoff's law, simple circuit analysis
    • transistors as switches
  • Analog/digital control (1 week)
    • ADC, DAC
    • PWM controllers
  • Electronics (1 week)
    • Op-amps
    • Linear devices
  • Applications (4 weeks)
    • Power applications - e.g., motor control
    • low power applications - battery operation
    • sensors
